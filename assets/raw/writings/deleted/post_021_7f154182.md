# Τρομακτικό 

Είναι τρομακτικό.

Το πως κάποια πράγματα όπως

Μια ιδέα

Ένα πρόσωπο

Ένα σύνολο ανθρώπων

Κάτι τέλος πάντων

Στο οποίο είχες και συνεχίζεις να επενδύεις

Μπορεί να σε πετάξει στον ουρανό και αμέσως μετά να σε πατήσει κάτω.

Και πάλι απ την αρχή.

Να έχεις πιστέψει ότι όλα θα είναι ροδινα,  να δίνεις,  δίνεις

Δίνεις δίνεις...

Και τελικά κάποια στιγμή να σκέφτεσαι,

Μήπως είμαι ηλιθιος ;  Μήπως δεν εχει νόημα;

Μήπως στερουμαι στιγμές για να τις δώσω κάπου, που δε θα εκτιμηθουν ;

Από την άλλη, τελικά έχει σημασία να εκτιμηθεί αυτό που προσφέρεις;

Μήπως το νόημα της ζωής είναι να δίνουμε ο, τι θέλουμε εμείς κι ας την πατάμε;

κι ας "χάνουμε"  χρόνο;

Γιατί ίσως, στο βάθος κάπου, εκεί

Να μην είναι χάσιμο.

Ίσως να είναι μάθημα- παράδειγμα προς αποφυγή για τις επόμενες πατάτες μας.

Ίσως είναι καμπανάκι που θα χτυπήσει όταν αργότερα τα πράγματα είναι πιο σοβαρά.

Γιατί πάντα υπάρχει "πιο"

Μα σκέψου δεν είναι τρομακτικό;

Κάνεις όνειρα,  βάζεις φαντασία,  μιλάς με τον εαυτό σου,  νιώθεις σίγουρος ότι θα πετύχεις και ότι κάνεις το σωστό.

Και τότε...

Μπουμ!

Σε χτυπάει η πραγματικότητα στη μούρη.

Με μια δόση σεισμού,  ο οποίος γκρεμιζει τη μαγεία που τόσο καιρό ένιωθες..

Σταγόνα σταγόνα

Εξαντλητικα αργά

Ώσπου θέλεις να ξεφυγεις

Ή απλά γυρίζεις σε αυτό που έκανες ,  ως αθώα ηλιθιος όπως λέγαμε πριν,  για να την ξαναπατησεις όμορφα και αγνά.

Σκέψεις της στιγμής.

Αγνοηστε το...
